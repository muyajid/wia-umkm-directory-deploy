# Grow UMKM  

**Grow UMKM** adalah sebuah website yang membantu pengguna untuk **menemukan bisnis usaha kecil di sekitar mereka** sekaligus **mempromosikan bisnis mereka sendiri** agar lebih dikenal oleh masyarakat luas.  
Website ini bertujuan untuk mendukung pertumbuhan UMKM lokal dengan memberikan wadah digital yang mudah digunakan, cepat, dan modern.  

---

## Tech Stack  
- **React JS + Vite**   
- **Tailwind CSS** 

---

## Cara Menjalankan Project  

1. Clone repository ini  
   ```bash
   git clone https://github.com/muyajid/wia-umkm-directory.git
2. Install Depedensi 
    ```bash 
    npm install 
3. Jalankan Server Development 
    ```bash 
    npm run dev 
4. Buka Web Di Browser 
    ```bash 
    http://localhost:5173
